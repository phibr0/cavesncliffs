Informationen

Die meisten von uns haben wahrscheinlich schon früher oft Minecraft gespielt. Um nun den Release der neuen Version "Caves and Cliffs" 1.18 zu feiern, haben wir einen Server erstellt. Wenn ihr möchtet, könnt ihr alle umsonst mitspielen, da der Server allerdings dennoch etwas kostet und wir auch viel Zeit in dieses Projekt investiert haben würden wir uns über kleine Spenden freuen, nutzt dazu am besten einfach den kleinen Knopf unten rechts in der Ecke!

Der Discord Server

Damit wir alle gemeinsam reden können, während wir spielen haben wir auch einen eigenen Discord erstellt. Es gibt genug Kanäle für alle Gruppen, wenn ihr mit euren Freunden einen eigenen haben wollt, könnt ihr den @Admins aber trotzdem bescheid sagen. Natürlich könnt ihr auch euren eigenen benutzen, hier werden aber auch wichtige Updates geteilt.

Auch hier gilt: Wer sich nicht benimmt, wird entfernt

Ein bisschen Trolling ist ok! :D

Backend

Da es die meisten nicht interessieren wird, ist dieser Teil ganz am Ende xD. Alle Dateien sind online in einem Mono-Repo auf Github zu finden. Diese Website wird via Github Pages gehosted und der Minecraft Server läuft in der Hetzner Cloud.

Repository Struktur:

            
├───docs            The Website
├───installer       Code for the Ressource Pack, Shader and Mod Installer
└───server          Server Code running on AWS Infrastructure
